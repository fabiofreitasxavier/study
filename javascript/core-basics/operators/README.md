### 1. Arithmetic Operators
Used for basic math.
- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `%` modulus (remainder)
- `**` exponentiation

```js
let a = 10, b = 3;
console.log(a + b);  // 13
console.log(a - b);  // 7
console.log(a * b);  // 30
console.log(a / b);  // 3.333...
console.log(a % b);  // 1
console.log(2 ** 3); // 8
