## Forms
- `<button>`: Button.
- `<datalist>`: Input suggestions.
- `<fieldset>`: Form control group.
- `<form>`: Form.
- `<input>`: Input control.
- `<label>`: Control label.
- `<legend>`: Fieldset caption.
- `<meter>`: Scalar gauge.
- `<optgroup>`: Option group.
- `<option>`: Select option.
- `<output>`: Form output.
- `<progress>`: Progress bar.
- `<select>`: Dropdown.
- `<textarea>`: Multi-line input.
