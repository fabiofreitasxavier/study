
## Formatting
- `<a>`: Hyperlink.
- `<abbr>`: Abbreviation.
- `<b>`: Bold (non-semantic).
- `<bdi>`: Bidirectional isolate.
- `<bdo>`: Bidirectional override.
- `<cite>`: Work title.
- `<code>`: Code fragment.
- `<dfn>`: Term definition.
- `<em>`: Emphasis.
- `<i>`: Italic (non-semantic).
- `<kbd>`: Keyboard input.
- `<mark>`: Highlighted text.
- `<q>`: Inline quote.
- `<s>`: Strikethrough.
- `<samp>`: Sample output.
- `<small>`: Small print.
- `<strong>`: Strong emphasis.
- `<sub>`: Subscript.
- `<sup>`: Superscript.
- `<u>`: Underline (non-semantic).
- `<var>`: Variable.
- `<wbr>`: Word break.
