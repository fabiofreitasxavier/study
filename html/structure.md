# Structures

- `<html>`: Document root.
- `<base>`: Base URL for relative links.
- `<head>`: Metadata container.
- `<link>`: External resource link.
- `<meta>`: Metadata (e.g., charset).
- `<style>`: Inline CSS.
- `<title>`: Document title.
- <!-->
- <!DOCTYPE HTML>
- `<body>`: Document content.
